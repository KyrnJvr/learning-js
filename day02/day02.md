Day 2 - Learning JavaScript

Objective: Learn how to declare and use variables

JavaScript has three kinds of variable declarations:
- `var` : declares a variable, optionally initialising it to a value
- `let` : declares a block-scoped, local variable, optionally initialising it to a value
- `const` : declares a block-scoped, read-only named constant (cannot change value)

I will mostly be using `let` and `const` in this course

NOTE: to easily understand `let` and `const`, remember that `const` value cannot be changed (hence the name constant) once initialised.

Example:
    let name = "John"; // name can be changed
    const myBirthday = '18-05-2000'; // you only have 1 and only 1 birthdate


Goal: 
1. Understand and differenciate `let`, `const`
2. Learn different data types (string, number, boolean, null, undefined)
    - string
