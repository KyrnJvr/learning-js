Day 1 - Learning JavaScript

Objective: understand what JS is and how it fits into web development

What is JS?
    - JavaScript is a high-level, interpreted programming language. It runs in the browser, but it can also run on servers via Node.js
    - It is used to make websites interactive (e.g. image sliders, pop-up messages, form validation)
    - It can Dynamically update content without reloading the page (via AJAX)
    - It can build full web applications (using frameworks like React, Vue, or Angular)
    - Can be used to develop mobile apps, games, and even desktop applications

Goals:
    1. Set up your development environment, and write your first JS script.


JavaScript can be inserted almost everywhere into an HTML document using the <script> tag:

<script>
    console.log("Hello World!");
</script>


You can also separate the js file from the HTML file so it's not too confusing:

<script src="path/to/your/script.js"></script>
