📘 30-Day JavaScript Learning Plan

Learn JavaScript in 30 days with this structured roadmap. Each day includes:

    🎯 Objective

    ✅ Goal

    📚 Resources

    🧠 Challenge

🗓️ Week 1: JavaScript Fundamentals

Day 1: Introduction to JavaScript

    🎯 Objective: Understand JavaScript’s role in the browser.

    ✅ Goal: Set up your environment and write a basic script.

    📚 Resources: MDN | JavaScript.info

    🧠 Challenge: Log “Hello, World!” and alert the current date and time using JavaScript.

Day 2: Variables & Data Types

    🎯 Objective: Learn to declare and use variables.

    ✅ Goal: Understand let, const, and primitive types.

    📚 Resources: MDN Variables | JS.info

    🧠 Challenge: Create variables for your name, age, and isStudent status. Log a sentence with them using template literals.

Day 3: Operators and Expressions

    🎯 Objective: Work with arithmetic and logical operators.

    ✅ Goal: Perform calculations and comparisons.

    📚 Resources: MDN Operators | W3Schools

    🧠 Challenge: Write a calculator that adds, subtracts, multiplies, and divides two numbers.

Day 4: Control Flow (if, else, switch)

    🎯 Objective: Use conditionals to control logic flow.

    ✅ Goal: Build a decision-making script.

    📚 Resources: JS.info | MDN Switch

    🧠 Challenge: Create a grading script that outputs a letter grade based on a test score (0–100).

Day 5: Loops

    🎯 Objective: Repeat tasks using loops.

    ✅ Goal: Master for, while, and do...while.

    📚 Resources: MDN Loops

    🧠 Challenge: Print all even numbers from 1 to 100 using a loop.

Day 6: Functions (Part 1)

    🎯 Objective: Write and reuse functions.

    ✅ Goal: Declare and call basic functions.

    📚 Resources: JS.info Functions

    🧠 Challenge: Write a function that takes two numbers and returns their sum.

Day 7: Mini Project & Review

    🧠 Challenge: Build a simple number guessing game.

        User guesses a number between 1–10.

        Program responds if it’s too high, low, or correct.

⚙️ Week 2: Intermediate JavaScript
Day 8: Functions – Scope, Parameters, Return

    🎯 Objective: Deepen function knowledge.

    ✅ Goal: Understand return values and scope.

    📚 Resources: MDN Functions

    🧠 Challenge: Write a function that calculates the factorial of a number.

Day 9: Arrays

    🎯 Objective: Work with collections of data.

    ✅ Goal: Store, access, and modify array elements.

    📚 Resources: JS.info Arrays

    🧠 Challenge: Create an array of 5 fruits. Add one, remove one, and log the result.

Day 10: Objects

    🎯 Objective: Store related data in an object.

    ✅ Goal: Create and access object properties.

    📚 Resources: JS.info Objects

    🧠 Challenge: Create an object representing a car (make, model, year). Log a descriptive sentence.

Day 11: Array Methods (map, filter, reduce)

    🎯 Objective: Transform arrays using functional methods.

    ✅ Goal: Use map(), filter(), and reduce().

    📚 Resources: FreeCodeCamp

    🧠 Challenge: Given an array of numbers, return the sum of only the even numbers.

Day 12: ES6+ Features

    🎯 Objective: Use modern JavaScript syntax.

    ✅ Goal: Practice arrow functions, destructuring, spread.

    📚 Resources: JS.info ES6

    🧠 Challenge: Rewrite a traditional function using arrow syntax. Use destructuring to extract object values.

Day 13: DOM Manipulation (Part 1)

    🎯 Objective: Interact with HTML using JavaScript.

    ✅ Goal: Select and modify DOM elements.

    📚 Resources: MDN DOM

    🧠 Challenge: Change the text of an HTML element by clicking a button.

Day 14: Project – To-Do List

    🧠 Challenge: Build a basic to-do list with add and remove functionality using the DOM.

🌐 Week 3: Events, APIs, Async JS
Day 15: Events and Event Listeners

    🎯 Objective: React to user interactions.

    ✅ Goal: Handle click, input, and submit events.

    📚 Resources: MDN Events

    🧠 Challenge: Build a form that alerts the input value on submit.

Day 16: DOM Manipulation (Part 2)

    🎯 Objective: Create and remove DOM elements dynamically.

    ✅ Goal: Add elements based on user input.

    🧠 Challenge: Build a dynamic list where items are added and removed with buttons.

Day 17: JSON & Local Storage

    🎯 Objective: Store and retrieve data locally.

    ✅ Goal: Use localStorage with JSON.

    📚 Resources: MDN Storage

    🧠 Challenge: Save a to-do list in localStorage and reload it on page refresh.

Day 18: Fetch API

    🎯 Objective: Get data from external sources.

    ✅ Goal: Make API calls using fetch().

    📚 Resources: MDN Fetch

    🧠 Challenge: Fetch and display a random user from https://randomuser.me/api.

Day 19: Promises & Async/Await

    🎯 Objective: Handle asynchronous code cleanly.

    ✅ Goal: Use async/await for fetch calls.

    📚 Resources: JS.info Promises

    🧠 Challenge: Fetch a random joke using async/await and display it.

Day 20: Error Handling

    🎯 Objective: Catch and handle runtime errors.

    ✅ Goal: Use try...catch.

    🧠 Challenge: Wrap your fetch from Day 19 in a try/catch and handle API errors gracefully.

Day 21: Project – Weather App

    🧠 Challenge: Build a weather app that fetches data from OpenWeatherMap or another public API.

🧠 Week 4: OOP, FP, Tooling, Final Project
Day 22: OOP Basics

    🎯 Objective: Understand classes and objects.

    ✅ Goal: Use constructor functions or class syntax.

    📚 Resources: JS.info OOP

    🧠 Challenge: Create a Person class with name and age, and a method to greet.

Day 23: Classes and Inheritance

    🎯 Objective: Extend and reuse class functionality.

    ✅ Goal: Use extends and super().

    📚 Resources: MDN Classes

    🧠 Challenge: Create a Student class that extends Person and includes a grade.

Day 24: Functional Programming Basics

    🎯 Objective: Write clean, functional code.

    ✅ Goal: Use pure functions and higher-order functions.

    📚 Resources: YouTube

    🧠 Challenge: Write a pure function that returns a new array of squared numbers from input.

Day 25: Modules & Imports

    🎯 Objective: Split JS code into modules.

    ✅ Goal: Use export and import.

    📚 Resources: MDN Modules

    🧠 Challenge: Refactor the weather app into modules: UI.js, API.js, main.js.

Day 26: Debugging & Tooling

    🎯 Objective: Use DevTools and linters.

    ✅ Goal: Set breakpoints and use console.log() effectively.

    📚 Resources: VSCode Debugging

    🧠 Challenge: Debug a small buggy function using Chrome DevTools.

Day 27: Basic Testing

    🎯 Objective: Write and run basic tests.

    ✅ Goal: Use console.assert() or Jest.

    📚 Resources: Jest Docs

    🧠 Challenge: Write a test for a function that returns the maximum of three numbers.

🏁 Days 28–30: Final Capstone Project
Build a Real-World JavaScript App

    🧠 Challenge: Choose one of the following projects (or create your own):

        ✅ Weather App

        ✅ Budget Tracker

        ✅ Quiz App

        ✅ Movie Search App using an API

    🎯 Goal: Implement everything you’ve learned: DOM, events, fetch, async, modules, and storage.
